Placez votre fichier de logo ici sous le nom `logo.png`.

Exemple :
- Copiez ou glissez votre image dans ce dossier
- Renommez-la en `logo.png`

Le site charge automatiquement `/logo.png` si présent. Après ajout, lancez :

```bash
npm install
npm run dev
```

Souhaitez-vous que j'enregistre l'image que vous avez jointe ici comme `public/logo.png` ? (répondez oui pour que je l'ajoute)
